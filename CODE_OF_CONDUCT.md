# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as our code of conduct. The full text is available at that link.

## Summary

In short: be respectful, be kind, give others the benefit of the doubt, and assume good intent. We welcome contributors from all backgrounds and experience levels. Disagreement is fine; personal attacks are not.

## Enforcement

Reports of misconduct can be sent to:

- **Private:** [GitHub Security Advisory](https://github.com/alea-drand/alea/security/advisories/new) (the same channel as vulnerability reports — mark the advisory as a conduct issue in the title)
- **Email fallback:** `security@alea.so`

Reports are reviewed and responded to on the same Tier A best-effort timeline as security reports (72h acknowledgement target, see `.github/SECURITY.md`). Confidentiality is preserved for reporters.
